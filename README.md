# Simple Todo App

A clean todo application built with vanilla JavaScript.

## Features

- Add, complete, and delete todos
- Optional due dates for todos
- Todo counter showing completed vs total
- Form validation
- Modal popup interface

## Technology

- HTML5, CSS3, JavaScript (ES6+)
- Component-based architecture
- BEM CSS methodology

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser

## Usage

- Click "+ Add Todo" to create a new todo
- Check the checkbox to mark a todo as completed
- Click the delete button to remove a todo
